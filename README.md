# python-cli-maven-scylladb-single-node-client

## Description
A POC for connecting single node scylladb with cql api to python.

## Tech stack
- python
  - scylla connector

## Docker stack
- scylladb/scylla
- python:3.8-slim

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
